# excel-abc_analysis-project
This project is an activity based consting spreadsheet for WellBeing Home Healthcare, designed to support internal decision making and provide managers with clear insights into the cost of serving different clients. 

Features include:
Use of formulae to automate calculations 
Provides results for clients A and B 


How to Use 
Open .xlsx link in Microsoft Excel
Review or update data in spreadsheet, formulaas will automatically recalculate results. 

Calculations follow the ABC methodology:
Allocate Resource Costs to Activities 
Allocate Activities to Cost Objects 
Combine Direct Costs and Overhead Allocations to caculate total cost
